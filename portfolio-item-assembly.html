AssemblyGame

React

GitHub Repo Live version

Assembly is a language-guessing game built with React, where players are challenged to identify programming languages based on unique clues. The game features over 500 hard-coded words and phrases, each tied to a specific language, creating a fast and engaging experience without the need for external APIs or data fetching.

This project showcases advanced state management, custom logic for win/loss conditions, dynamic timers, and accessibility features. React hooks like useEffect, useState, and useRef are used extensively to manage game flow and user interaction.

Assembly focuses on clean UI, quick performance, and a fun learning.
