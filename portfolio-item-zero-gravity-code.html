This is Zero Gravity Code, a personal website and freelance portfolio designed to showcase skills, projects, and services in a clean, professional, and accessible manner. Built with vanilla JavaScript and modern CSS techniques, it emphasizes performance, responsiveness, and user experience.

The site features a mobile-first, fully responsive layout that adapts seamlessly to different screen sizes, ensuring users can access it on any device. Key sections are organized for clarity, with a clean navigation system and visually appealing content presentation.

Above the fold, a starry sky animation is drawn and animated using the HTML canvas element, adding a visually engaging and dynamic element to the user experience.

Performance and accessibility are core priorities: the website achieves high Google PageSpeed scores, follows on-page SEO best practices, and adheres to a11y standards, making it inclusive and search-engine friendly.

This site demonstrates strong front-end development fundamentals, attention to detail, and a focus on creating professional, client-ready web experiences.
