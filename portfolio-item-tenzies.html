TenziesGame

React

GitHub Repo Live version

This is Tenzies, a fast-paced dice game built with React, where the objective is to roll until all dice show the same number. Players can “hold” dice to freeze their values between rolls, encouraging strategy and quick thinking. This version includes a real-time timer, roll counter, and animated confetti celebration when the player wins.

This project highlights essential React concepts like component-driven design, hooks (useState, useEffect, useRef), and conditional rendering. The layout is styled with responsive CSS and prioritizes accessibility and simplicity to keep the gameplay intuitive and enjoyable.

Tenzies is a fun way to demonstrate interactivity and state management in a clean, lightweight React app.
