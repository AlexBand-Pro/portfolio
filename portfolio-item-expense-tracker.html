ExpenseTracker

JavaScript | Chart.js

GitHub Repo Live version

This is a browser-based application for tracking and visualizing personal expenses, designed to help users manage their spending in a clear, interactive, and accessible way. Built with vanilla JavaScript, HTML, CSS, and Chart.js, it emphasizes performance, responsiveness, and usability.

Users can add and remove expense entries, each including item name, amount, category, and date. The application calculates total expenses in real-time and automatically updates a dynamic chart to provide instant visual feedback. All data is stored locally using localStorage, ensuring persistence across sessions.

The app features a clean, responsive layout, optimized performance, and enhanced accessibility (a11y), making it usable across devices and inclusive for all users.

This project demonstrates strong front-end development skills, including data management, DOM manipulation, chart integration, accessibility best practices, responsive design, and client-focused user experience.
